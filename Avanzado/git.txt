Repo local
git init 			-> inicializa el repositorio
git add . 			-> Pasando todos los archivos al staging area
git status 			-> Ver archivos trackeados
git commit -m ""	-> Registrando cambios en el repo, -m agrega un comentario
git log				-> Registro de commits
git checkout hash 	-> Regresas en el tiempo
git checkout master	-> Regresas a la ultima version
